# Find AlphaZero API Files

Header files for {{cookiecutter.game_name}} implementation are in the[include](../include/) directory.

API interface files in the [AlphaZero API](https://github.com/shuyangsun/alpha-zero-api) can be found by first building the project, then looking into `build/debug/_deps/alphazeroapi-src` directory. Header files are likely in `build/debug/_deps/alphazeroapi-src/src/include/alpha-zero-api`.
